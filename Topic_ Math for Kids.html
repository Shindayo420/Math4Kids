Here's what we are building

 

  1. Get the 9 Math for Kids handouts from teacher
  2. Make a Math for Kids Folder on your computer
  3. Save the audio file below inside that Math for Kids folder 
  4. Follow instructions on the handout
  5. Open Visual Studio Code
  6. Make your 9 files
    1. index.html
    2. subtract.html
    3. divide.html
    4. multiply.html
    5. add.js
    6. subtract.js
    7. divide.js
    8. multiply.js
    9. style.css
  7. You will need to copy the font awesome code from the font awesome site
  8. Publish work in Github when finished
  9. Submit file here in Canvas
  10. This is a summative assessment grade
  11. Change the colors up to customize it and make it your own unique project
  12. Do not share github codes with anyone else (That's cheating!)
  13. However, you can help each other out within Visual Studio Code
  14. Be sure to split your screen, go Live to see your site in the browser on one side along with your code on the other side.
